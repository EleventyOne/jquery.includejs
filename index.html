jquery.includejs

Simple jQuery plugin that allows you to dynamically load underscore templates and javascript files.

View the Project on GitHub EleventyOne/jquery.includejs

This is a straightforward plug-in that allows you to dynamically load templates (for underscore, defined as <script type='text/template'> blocks) and .js files. It is not like RequireJS where the loaded .js files are treated as modules. It is understood that what you load will be defined in the global namespace.

Personally, I used it to avoid having to load all of my Backbone.View files at once on the main page. This way I can spread them out to where they are needed.

Note that it does NOT load .css files, as that turned out to be a nasty mess to get working, so you will have to do that manually on the main page for all .js files that require it.

Example:

template.view.menu.tpl.htm

<script type="text/template" id="tpl-menu">
  ... content ...
</script>

view.menu.js

// this only highlights the code relevant to IncludeJS, obviously
// much more is required to write a proper Backbone.View

var MenuView = Backbone.View.extend({

  ... content ...

  initialize : function(...) {
    this.templates["tpl-menu"] = $.includejs.getTemplate("tpl-menu",{});
  },

  render : function() {
    this.$el.html(this.templates["tpl-menu"]);
    return this;
  }

});

app.js

// settings should be initialized once only, except for perhaps the cache parameter.

$.includejs.settings.ROOT = "";
$.includejs.settings.TPL_PATH = "_inc/tpl/";
$.includejs.settings.JS_PATH = "_inc/js/";
$.includejs.settings.cache = false;

// called everytime you want to stream in one or more files.

$.includejs.include({
  
  tpl : ["template.view.menu"],
  js : ["view.menu"],
  
  waiting : function() {
    console.log("Loading template(s) and javascript file(s)...");
  },

  success : function() {
    view = new MenuView(...);
    view.render();
  },

  error : function(options,jqXHR,textStatus,errorThrown) {
    console.log("Failed.");
  }
});

... code continues, callbacks executed when appropriate...
